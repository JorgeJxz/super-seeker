# Super Seeker 🧠

Aplicación full-stack de análisis de noticias usando NLP.

## Stack

### Backend
- Python
- FastAPI
- Pandas
- NLTK
- NewsAPI

### Frontend
- React
- Chart.js

## Features

- Búsqueda de noticias por tema
- Análisis de frecuencia de palabras
- Bigramas
- Análisis de sentimiento
- API REST propia

## Run backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
